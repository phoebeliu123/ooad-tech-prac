# Poker Game (OOAD Project 01)

這是一個基於物件導向設計（OOAD）實作的撲克牌比大小遊戲。支援人類玩家與 AI 玩家進行對戰，並包含特殊的「交換手牌」機制。

## 遊戲規則

1. 遊戲準備

- 遊戲共有 4 名玩家，包含 2 名人類玩家（Human）與 2 名 AI 玩家。

- 使用一副標準撲克牌（52 張），涵蓋四種花色：黑桃 (SPADES)、紅心 (HEARTS)、方塊 (DIAMONDS)、梅花 (CLUBS)。

- 每位玩家會依序抽牌，直到手牌滿 13 張為止。

2. 回合流程

- 出牌機制：每一輪中，有手牌的玩家需從手中選擇一張牌打出。

- 交換手牌：每位玩家在遊戲中都有一次主動發起「交換手牌」的機會。

發起交換後，雙方手牌會立即對調。

- 交換期限：交換狀態會持續 3 回合，第 3 回合結束後，手牌會自動換回給原主。

- 限制：只能挑選目前未處於交換狀態的玩家進行交換。

3. 勝負判定

- 單輪勝負：比較該輪玩家打出的牌。點數較大者獲勝；若點數相同，則比較花色。

- 點數順序：A(14) > K(13) > ... > 2(2)。

- 花色順序：黑桃(4) > 紅心(3) > 方塊(2) > 梅花(1)。

- 積分與終局：單輪獲勝者可得 1 分。當所有玩家手牌出清時遊戲結束，總分最高者為最終贏家。

## 如何執行

1. 確保您的環境已安裝 Python 3。

2. 下載此專案原始碼。

3. 在終端機（Terminal/Command Prompt）進入專案目錄下的 p1-code 資料夾。

4. 執行以下指令啟動遊戲：

```
python3 main.py
```
